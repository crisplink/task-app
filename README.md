# My Task Manager

A personal daily task manager built with vanilla HTML, CSS and JavaScript.

## Features
- Add, edit and delete tasks
- Set priority — High, Medium, Low
- Set reminders with a clock picker
- Today view and History view grouped by date
- Progress bar with completion animation
- Dark and light mode
- Works fully offline after first visit
- Tasks saved locally on your device

## Install on Phone
1. Open the live site in Chrome (Android) or Safari (iPhone)
2. Tap Share → Add to Home Screen
3. Opens and works like a native app
